## 🏆 HACS integrations (Home Assistant)

- **[home-assistant-sikom](https://github.com/Howard0000/home-assistant-sikom)**  
  [![HACS](https://img.shields.io/badge/HACS-Default-green.svg)](https://hacs.xyz/)  
  A **HACS integration for Home Assistant** for Sikom.  
  Simple UI configuration, automatic device discovery, and support for thermostats, switches, AMS meters, and Easee chargers.

- **[home-assistant-norsk-tidevann](https://github.com/Howard0000/home-assistant-norsk-tidevann)**  
  [![HACS](https://img.shields.io/badge/HACS-Default-green.svg)](https://hacs.xyz/)  
  A **HACS integration for Home Assistant** for tidal data from the Norwegian Mapping Authority.  
  Includes sensors and visualizations (ApexCharts) for an easy overview in Home Assistant.

- **[home-assistant-systemair-modbus](https://github.com/Howard0000/home-assistant-systemair-modbus)**  
  [![HACS](https://img.shields.io/badge/HACS-Default-green.svg)](https://hacs.xyz/)   
  A **custom Home Assistant integration** for Systemair SAVE units via Modbus TCP.  
  Provides structured monitoring and control of ventilation, with a focus on stable entity handling and energy-efficient operation.

- **[home-assistant-hav-og-vind](https://github.com/Howard0000/home-assistant-hav-og-vind)**  
  [![HACS](https://img.shields.io/badge/HACS-Default-green.svg)](https://hacs.xyz/)   
  A **custom Home Assistant integration** for wave and wind data.  
  Provides sensors and visualizations for maritime conditions and weather data.
